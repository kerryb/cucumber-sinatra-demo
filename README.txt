Dead simple demo project with Cucumber features running against a simple
Sinatra app.

Prerequisites:

Ruby, RubyGems, and some gems:

  $ gem in sinatra rspec rack-test cucumber webrat

Running the tests:

  $ cucumber

Running the sinatra app (not required to run the tests):

  $ ruby app.rb # appears on http://localhost:5678

More info:

Sinatra:  http://www.sinatrarb.com/
Cucumber: http://cukes.info/
Webrat:   http://gitrdoc.com/brynary/webrat/tree/master/
